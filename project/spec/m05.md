# M05: Agent Expansion

## Objective

Extend the identity, detection, and case surfaces to service accounts and agents without breaking the existing human-focused model.

## User-Visible Outcome

Analysts can investigate human and non-human principals in a single evidence-backed workflow.

## Required Capabilities

- non-human principal representation
- agent and tool-invocation aware actions
- mixed human and agent timelines
- scoring and evidence retrieval that operate across principal types

## Acceptance Criteria

- service accounts and agents are first-class principals
- mixed human and agent cases can be created and resolved
- no separate bespoke pipeline is required for the new principal classes

## Dependencies

- `m01`
- `m02`
- `m03`

## Explicitly Out Of Scope

- full agent governance platform
- generalized agent orchestration or model hosting
- broad expansion into non-security agent observability
